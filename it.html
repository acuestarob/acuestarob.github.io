Project Ideas

Overveiw of Project

The project idea that I have come up with involves the use of our smartphones. It's an app which can pretty much make our lives easier with everything assesible through our finger tips. It's an app designed to control our cars (being able to open/lock door and turn/off car) and control our houses (light switches, tv's, heater/airconditioner, etc.) all through an app.

Motivation

I beleive that the project idea will be useful as we are always on our smartphones, and just being able to access our cars/house without bringing around extra keys around the place and being able to control certain things in our household without the use of remotes, swtiches, etc through our smartphones can make life more convenient.

Description
You can easily transform your smartphone into a multipurpose based devise in which you can easily access your house/car aswell as household items/appliances with a single touch.

Key Features:

-Manage all connections in one screen

-Easy connection to multiple devices

-Key combination input

-Multi-Touch Gestures

-Works under 3G/4G and Wi-Fi network

-Voice control (Siri)

Tools and Technologies

Appery.io is a great software to use as it is a cloud-based platform that provides UI development tools and integrated backend services. You can build hybrid web apps for Android and iOS.

Appery.io provides a complete backend for your app:

-Database - for storing app data

-Server Code - for writing app logic on the server using JavaScript

-API Express - exposing enterprise data sources such as an SQL database or SOAP service via REST APIs. Also, build advanced services with a visual service editor

-Push Notifications - send Push Notifications to your users

Skills Required

Skills required include; UX/UI Design Skills, Modern Programming Languages (HTML, Objective-C, Java, etc.), Agile Methodologies, Cross-Platform Knowledge and must think like a USER.
